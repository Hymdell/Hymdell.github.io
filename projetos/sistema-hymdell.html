Sobre o projeto

O Sistema Hymdell é um sistema de chamados desenvolvido para organizar e registrar as manutenções que realizo em computadores e notebooks na Projeção Informática.

O sistema permite abertura, acompanhamento e encerramento de chamados, com registro do cliente, equipamento, problema relatado e solução aplicada. Foi construído com PHP no back-end, banco MySQL e interface em HTML, CSS e JavaScript.

A estrutura do projeto é dividida em frontend/ e backend/, mantendo uma separação clara de responsabilidades mesmo sendo um projeto solo.

Funcionalidades

Abertura e registro de chamados com dados do cliente e equipamento, acompanhamento do status de cada chamado, registro da solução aplicada e histórico de manutenções por cliente.
