Sobre o projeto

O Sistema de Férias foi desenvolvido durante o estágio na Quero-Quero como solução para o processo de solicitação e aprovação de férias dos funcionários.

Funcionários podem registrar suas solicitações de férias pelo sistema, informando o período desejado. Gestores têm acesso a um painel de aprovação onde podem aprovar ou rejeitar as solicitações, além de gerenciar as informações da equipe.

O sistema foi construído com PHP no back-end integrado a um banco MySQL, com interface web em HTML, CSS e JavaScript.

Funcionalidades

Solicitação de férias pelo funcionário com escolha de período, painel do gestor para aprovar ou rejeitar solicitações, gerenciamento de equipe e histórico de solicitações por funcionário.

Tecnologias

HTML JavaScript PHP MySQL
